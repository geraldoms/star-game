# React Star Game

=======================

This is a basic react star game when you need to select one or more numbers whose the sum matches the number of stars in 10 seconds.

## Requirements

- Node 8+

## Installation

`$ npm i`

## Usage

`$ npm start`

Usage example

When the application starts you can access it at `http://localhost:3000/`.
